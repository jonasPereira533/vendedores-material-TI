# 🛒 Sistema de Vendas em Java

Projeto desenvolvido para praticar conceitos de programação em **Java**.  
O sistema permite cadastrar vendedores, registrar vendas, calcular automaticamente as comissões e gerar relatórios com totais.

---

## 📌 Funcionalidades
- Cadastro de vendedores
- Registro de vendas com valor e percentual de comissão
- Cálculo automático do valor da comissão em reais
- Relatório detalhado de cada vendedor
- Exibição do total de vendas e do total de comissões

---

## 🛠️ Tecnologias utilizadas
- **Java** 
- Programação Orientada a Objetos
- Programação funcional 
- Coleções (`ArrayList`)
- Estruturas de repetição (`while`, `for`)
- Entrada de dados via `Scanner`
